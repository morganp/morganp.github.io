"Bash String Manipulation"

Published: Wed 05 March 2014
By morganp

In posts.

  • Command Line
  • bash

Removal of substring:

string="test string"
echo $string

remove="test"
short_string=${string#$remove}
echo short_string

Note that inside the ${} string does not have a $, it is implied. The second half $remove does require it.
