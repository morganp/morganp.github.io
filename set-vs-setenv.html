"set vs setenv"

Published: Fri 11 October 2013
By morganp

In posts.

  • Command Line
  • Programming discuss_url: //236 url: //236/set_vs_setenv id: 236

setenv allows sub shells to inherit the value.

Set

set x = "twenty"
echo $x
> twenty

## New Shell
csh
echo $x
x: Undefined variable.

Setenv Note missing '='

setenv x "twenty"
echo $x
> twenty

##New Shell
csh
echo $x
> twenty
