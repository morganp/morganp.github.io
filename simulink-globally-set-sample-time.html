"Simulink: Globally set sample time"

Published: Wed 30 July 2014
By morganp

In posts.

To set each constant in a model to sample time -1:

blks = find_system(dut, 'BlockType', 'Constant');
for i = 1:length(blks)
    set_param(blks{i}, 'SampleTime', '-1');
 end
