"Stop/Start and Restart Samba on OS X"

Published: Sun 10 October 2010
By morganp

In posts.

To Stop Start and Restart Samba on Mac OS X 10.6:

sudo /usr/sbin/smbd stop
sudo /usr/sbin/smbd start
sudo /usr/sbin/smbd restart
