"Nginx Enable Directory Listing"

Published: Mon 10 May 2010
By morganp

In posts.

  • Web
  • nginx discuss_url: //30 url: //30/Nginx_Enable_Directory_Listing id: 30

Directory listing can be enabled (disabled by default) by adding this:

autoindex on;

Example usage

http{
   server {
     #Server with no Directory listing
     listen 80
   }
   server {
      #Server with Directory listing
      autoindex on;
      listen 8080
   }
}
