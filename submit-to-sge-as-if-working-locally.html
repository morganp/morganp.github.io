"Submit to SGE as if working locally"

Published: Wed 30 July 2014
By morganp

In Engineering.

tags: Programming SGE

Setting an alias in you .tcshrc

alias qrun 'qrsh -V -noshell -cwd !*'

or bash 

alias qrun='qrsh -V -noshell -cwd !*'

Which makes submitting any script to the grid just:

qrun helloworld.sh
