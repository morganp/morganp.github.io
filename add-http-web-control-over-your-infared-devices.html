"Add HTTP (web) control over your InfaRed devices"

Published: Tue 11 May 2010
By morganp

In Tech.

tags: Audio Web Automation

Using the new IR ANYWHERE IR OVER IP MODULE you can send web requests that get turned into standard consumer InfaRed signals, to control your tv, hifi units. I have thought for along time that manufacturers should just be adding web control to all new electronics, but this seems like it could be a good interim measure.

This then allows app/web based control of you home entertainment system, which would be very slick from an iPhone/iTouch/iPad.
